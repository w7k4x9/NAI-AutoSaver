# NAI 자동저장

NovelAI 이미지 페이지(`https://novelai.net/image`)에서 작동하는 Chrome Manifest V3 확장입니다.
자동 생성 · 대기열 · 조합 생성 · 제목 기반 자동 저장을 지원합니다.

## 핵심 기능

- **자동 생성**: 지정한 횟수만큼, 지정한 주기(초)로 NovelAI `Generate` 버튼을 자동 클릭
- **완료 이미지 자동 저장 (기본 ON)**: 생성될 때마다 이미지를 자동 다운로드
- **제목 기반 저장 파일명**: 저장 이름이 `1_b_c`면 `1_b_c (1).png`, `1_b_c (2).png` … 순서로 저장 (NovelAI 폴더 아래)
- **대기열 (Base / 캐릭터 프롬프트 분리)**: 서로 다른 (저장 이름 · Base Prompt · 캐릭터 프롬프트 · 횟수) 항목을 줄 세워 무인 자동 생성
  - `현재 프롬프트 추가`: 지금 NovelAI의 Base 프롬프트 + 캐릭터 프롬프트 + 저장 이름 + 횟수를 항목으로 저장
  - 항목마다 **저장 이름 · Base Prompt · 캐릭터 프롬프트 · 횟수**를 따로 입력/편집, 순서 변경(↑↓), 삭제
  - **Base는 베이스 칸, 캐릭터는 NovelAI 캐릭터 박스에 직접 입력됩니다.** (`|` 병합 방식은 캐릭터 박스가 이미 있으면 NovelAI가 무시하므로 폐기) 캐릭터 박스가 모자라면 `+ Add Character`를 자동으로 눌러 추가합니다.
  - 한 항목에 여러 캐릭터는 캐릭터 칸에서 `|`로 구분 → 박스 1·2·…에 순서대로 입력
  - `대기열 실행` → 항목1의 Base/캐릭터 적용 → 횟수만큼 생성·저장 → 항목2 적용 → … 자동 진행
  - **저장 이름은 항목마다 따로 적용됩니다.** ‘자동 생성’ 섹션의 저장 이름은 단발 실행 전용이며 대기열에는 적용되지 않습니다.
  - 펼친 대기열은 **가로 스크롤 카드(위) + 큰 편집창(아래)** 구조입니다. 카드를 누르면 아래에 저장 이름 · Base Prompt · 캐릭터 프롬프트를 큼직한 입력란으로 편집할 수 있고, 실행 중인 항목은 자동으로 보이게 스크롤됩니다
  - 항목별로 저장 파일명이 그 항목의 저장 이름으로 `(1)`부터 매겨짐
  - `반복` 켜면 끝까지 돈 뒤 처음부터 다시 반복
  - 모든 항목 완료 시 알림·사운드, 대기열 JSON `내보내기`/`가져오기` (구버전 단일 프롬프트 파일은 자동으로 Base 프롬프트로 변환되어 들어옴)
- **조합 생성기**: Base 1개, 캐릭터 N개(코드·외형·표정세트), 배경 M개(코드·배경태그)를 입력하면 모든 (캐릭터 × 배경 × 표정) 조합을 대기열 항목으로 자동 생성합니다. 항목 제목은 `캐릭터코드_배경코드_감정번호`(예: `w_h_1`), 캐릭터 프롬프트는 `외형 / 표정 / 배경` 3블록으로 구성됩니다. `현재 큐에 추가` · `새 큐로 교체` · `JSON 내보내기`(파일당 최대 항목 수로 분할 가능)를 지원하며, 표정 줄 앞 번호는 자동 제거됩니다.
- **iOS 리퀴드 글래스 UI**: 반투명 글라스 패널 · 광택 하이라이트 · 글라스 토글 (드래그 이동 · 접기/펴기), 글라스 앱 아이콘

## 사용법

1. NovelAI 이미지 페이지 오른쪽 아래에 패널이 뜹니다. (작은 알약을 누르면 펼쳐짐)
2. 프롬프트는 NovelAI 자체 입력칸에서 작성합니다.
3. 단발 자동 생성: `저장 이름`, `횟수`(0=무제한), `주기(초)`를 정하고 `자동 생성 시작`.
4. 대기열: `현재 프롬프트 추가`로 항목을 만든 뒤, 각 항목 카드에서 **저장 이름 · Base Prompt · 캐릭터 프롬프트 · 횟수**를 따로 채웁니다 → `대기열 실행`. (Base/캐릭터를 비우면 추가 당시 NovelAI 프롬프트가 사용됩니다)
5. 멈추려면 `자동 생성 중지` / `대기열 중지`, 또는 접힌 패널의 ■ 버튼.

저장 위치는 브라우저 다운로드 폴더의 `NovelAI/` 하위입니다. 같은 이름이 있으면 Chrome이 자동으로 번호를 덧붙입니다.

## 로드 방법

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 개발자 모드를 켭니다.
3. "압축해제된 확장 프로그램을 로드"를 누릅니다.
4. 이 폴더를 선택합니다.
